Qn2. Controlling colour of RGB LED and blinking speed of an
other LED with potentiometer

RGB LED Control using Potentiometer:

I built this circuit using a potentiometer to control
both an RGB LED and a normal LED. Initially, I explored 
how analog input works and how it can be mapped to different 
outputs. By adjusting the potentiometer, I was able to vary
the colour of the RGB LED using PWM. At the same time, I used
the input value to change the blinking speed of another LED. 
It took some trial and error to balance both controls properly,
but I understood how analogRead and analogWrite work together through this task.
