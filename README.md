MaRS Electronics Task 1

This repository contains my work for the MaRS Electronics Task 1 
using Arduino and Tinkercad. I started with very little prior knowledge
and learned the required concepts through online resources such as YouTube and AI tools.
Through these tasks, I explored concepts like millis() for timing, PWM for RGB LED control,
and basic sensor interfacing using LDR and ultrasonic sensors. 
I also built mini-projects such as an Automatic Night Lamp and a Smart Parking Assist System.
All circuits were tested through simulations and their outputs are included.
