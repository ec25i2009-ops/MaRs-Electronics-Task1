Question 3

This folder contains the implementation of two Arduino-based mini projects developed using Tinkercad:
1. Automatic Night Lamp (LDR):
This project uses a photoresistor (LDR) to detect ambient light levels.
The system automatically turns an LED ON in low-light (dark) conditions
and turns it OFF in bright conditions.

2. Smart Parking Assist System:
This project is inspired by an ultrasonic distance alert system.
It uses an ultrasonic sensor to measure distance and multiple LEDs
of different colours to indicate proximity levels. As an object gets
closer, different LEDs glow to alert the user, simulating a real-world parking assist system.
